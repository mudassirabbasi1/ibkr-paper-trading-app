# IBKR Paper Trading App

A Python-based paper trading app using the Interactive Brokers API (via ib_insync).

## Features
- Connects to IBKR TWS / Gateway
- Supports paper trading for stocks
- Generates trade execution reports
- Built with Python and ib_insync

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
